Security Points¶

  1. You are uploading the file use below code. $file = basename(realpath($_GET['file']));
  2. Avoid using mysql(i)_ extensions use PDO filter_var($_REQUEST['search'], FILTER_SANITIZE_FULL_SPECIAL_CHARS);
  3. use password_hash and password_verify $hash = password_hash($password, PASSWORD_DEFAULT); password_verify($password, $hash);
  4. In php.ini change expose_php to Off
  5. In php.ini change mail.add_x_header to Off
  6. In php.ini change session.cookie_httponly to On session.cookie_httponly = On
  7. In php.ini change session.cookie_secure to On session.cookie_secure = On
  8. In php.ini change session.use_strict_mode to On session.use_strict_mode = 1
  9. allow_url_fopen = Off allow_url_include = Off
  10. In php.ini change open_basedir open_basedir = /var/www/html/ // Multiple directories can be specified with the ":" delimiter open_basedir = /var/www/html/:/var/www/html2/:/var/www/html3/
  11. In php.ini change upload_temp_dir upload_tmp_dir = /var/www/html/tmp/ filter_var($email_address, FILTER_VALIDATE_EMAIL)
  12. filter_var($ip_address, FILTER_VALIDATE_IP)
  13. filter_var($url, FILTER_VALIDATE_URL)
  14. Don’t Do This <a href="http://example.com" target="_blank">Click here</a> Do This Instead <a href="https://example.com" target="_blank" rel="noopener noreferrer">Click here</a>

SSL Redirection¶

##Redirection to https file**
RewriteCond %{HTTPS} off
RewriteRule .* https://%{HTTP_HOST}%{REQUEST_URI} [L,R=301]

##Redirection to https and www page**
RewriteCond %{HTTP_HOST} !^www\. [NC]
RewriteRule .* https://www.%{HTTP_HOST}%{REQUEST_URI} [L,R=301]
